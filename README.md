# Multi-Agent Systems Research

Deep-dive research on multi-agent systems used in academia and industry — frameworks, architectures, comparisons, and a structured learning path for engineers.

## Live Documentation

**[https://pablordoricaw.github.io/multi-agent-systems-research/](https://pablordoricaw.github.io/multi-agent-systems-research/)**

## Contents

- **Executive Overview** — Key trends, dominant frameworks, and open problems in multi-agent systems
- **Getting Started** — Minimal viable stack, key concepts (memory, tool use, orchestration), and code examples
- **Deep Dives** — One page per major framework: LangGraph, AutoGen, CrewAI, OpenAI Agents SDK, Agency Swarm, SWE-agent, OpenHands, Aider, Patchwork, Devin
- **Comparison Tables** — Side-by-side framework analysis with decision matrix and maturity assessment
- **Learning Path** — Opinionated 4-week curriculum from single-agent tool use to production multi-agent pipelines
- **Changelog & Sources** — 20+ academic papers, 11 GitHub repositories, and web sources with full citations
